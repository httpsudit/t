Here's an example of a more complex HTML code that includes a header, navigation bar, main content area, and a footer:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
        }
        header {
            background-color: #f0f0f0;
            padding: 20px;
            text-align: center;
        }
        nav {
            background-color: #333;
            padding: 10px;
            text-align: center;
        }
        nav a {
            color: #fff;
            text-decoration: none;
            margin: 10px;
        }
        main {
            display: flex;
            flex-direction: column;
            align-items: center;
            padding: 20px;
        }
        footer {
            background-color: #f0f0f0;
            padding: 10px;
            text-align: center;
            clear: both;
        }
    </style>
</head>
<body>
    <header>
        <h1>My Website</h1>
    </header>
    <nav>
        <a href="#">Home</a>
        <a href="#">About</a>
        <a href="#">Contact</a>
    </nav>
    <main>
        <h2>Welcome to My Website</h2>
        <p>This is a paragraph of text on my website.</p>
        <button>Click Me</button>
        <ul>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ul>
        <img src="image.jpg" alt="An image on my website">
        <a href="https://www.example.com">Visit Example Website</a>
    </main>
    <footer>
        <p>&copy; 2024 My Website</p>
    </footer>
</body>
</html>
```

This code includes:

* A `header` element with a heading and background color
* A `nav` element with links to different pages
* A `main` element with a heading, paragraph, button, unordered list, image, and link
* A `footer` element with a copyright notice and background color
* CSS styles to layout and design the elements

You can save this code as an HTML file and open it in a web browser to see the result.
